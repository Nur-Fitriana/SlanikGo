<h1 align="center">🌊 SlanikGo</h1>

<p align="center">
SlanikGo merupakan website yang menyediakan informasi tentang Slanik Waterpark di Lampung Selatan. 
Website ini menampilkan informasi seperti fasilitas, harga tiket, jam operasional, lokasi, dan foto wisata. 
Dengan adanya SlanikGo, pengunjung dapat lebih mudah mengetahui informasi tentang Slanik Waterpark sebelum berkunjung.
</p>
